# ======================
# Home-made video server
# ======================


* Pre-requisites
  - Python 3
  - ffmpeg

    
* Add videos
  1. mkdir videos
  2. place .mp4 videos here

     
* Set up server
  1. python -m venv venv
  2. . venv/bin/activate
  3. pip install flask

     
* Run server
  ./run.sh
